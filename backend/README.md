### Docker
* docker pull quycuteprovip/tornado-app:v1.0
* docker run -p 8888:8888 quycuteprovip/tornado-app:v1.0
### Python
* pip install -r requirements.txt
*  python app.py
*  Open webrowser: [localhost:8888](http://localhost:8888/)
